Description du Projet :

Dans le contexte M2L, la maison des ligues a décidé de publier les données relatives aux résultats des matchs de la fédération lorraine de FutSal (football en salle), et ce, via une application Web. Elle souhaite pour cela rassembler les données de ces résultats dans une base de données MySql, avant de passer à la phase de développement de cette application.


L’objectif de ce projet est d’obtenir une base de données MySql, ayant la même structure que la base Access FLFS.mdb existante, et remplie avec les données contenues dans les fichiers texte existants.


Les outils utilisés :

  • Langage de Programmation : C#
  • IDE : Visual Studio
  • Outil de Versionning : Git
  • Site de versionning : GitLab (Il faut créer un compte)
  • Type de projet C#: Windows Forms

Compétences acquises :
