Description du Projet :

Ce projet explique les essentiels du JDBC (Java DataBase Connectivity), de JavaFX et comment le modèle MVC (Modèle Vue Controleur) est écrit.


JavaFX est un framework de Java qui permet aux développeurs de créer une interface graphique pour les applications de bureaux.


JDBC est un API pour Java qui permet à un utilisateur de se connecter avec la base de données sous architecture de MVC.

Les outils utlisés :

  • Langage de programmation : Java
  • IDE : JetBrains Intellij Idea (Version communauté)
  • Outil de Versionning : Git
  • Site de Versionning : GitHub (Il faut créer un compte)
  • Frameworks utilisé : JavaFX
