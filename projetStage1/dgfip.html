Description du Projet :

METRIS est un projet appartenant à la DGFIP. Il sert à être utilisé pour la métrologie des modules et des applications.


Atlas est l'un des modules présents dans le projet. Il sert à récupérer la volumétrie des documents pour chaque application, le nombre total de consultations pour chaque applications, l'évolution des deux, etc.


On est demandé de moderniser le module Atlas en utilisant Java, JavaScript et JSP, tout en s'inspirant du code des autres modules existants.

Les outils utilisés:

  • Langages utilisés :
    • Java
    • JavaScript
    • JSP (Jakarta Server Pages)
  • IDE utilisé : Eclipse
  • Frameworks utilisé:
    • JQuery (Pour Javascript)

Les compétences acquises :

  • Gérer le patrimoine informatique:
    • Recenser et identifier les ressources numériques
    • Exploiter des référentiels, normes et standards adoptés par le prestataire informatique
  • Travailler en mode projet:
    • Planifier les activités
