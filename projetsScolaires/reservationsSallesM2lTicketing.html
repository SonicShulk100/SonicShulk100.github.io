Réservation Salles M2L - Ticketing

Description du projet :


Ce projet explique comment fonctionnent les tickets. Le ticketing est un ensemble de tickets décrits par les besoins des clients.


Les outils Utilisés :

  • Langage de Programmation : C#
  • IDE : Visual Studio
  • Outil de Versionning : Git
  • Site de Versionning : GitLab (Il faut créer un compte)

Compétences acquises:

  • Répondre aux incidents et aux demandes d’assistance et d’évolution
    • Collecter, suivre et orienter des demandes
    • Traiter les demandes concernant les applications
  • Travailler en mode projet:
    • Planifier les activités
    • Analyser les objectifs et les modalités d’organisation d’un projet
