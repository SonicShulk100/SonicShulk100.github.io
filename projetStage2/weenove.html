Editeur de fichiers ODT en PHP pure

Description du Projet :


Sujet :


Développement d'un éditeur de fichiers ODT qui va être intégré au logiciel no-code Weemake


Contexte :


Weemake est une plateforme type "no-code" qui permet de créer des applications métiers sur mesure.


Weemake propose un module "documents" qui génère des fichiers PDF à partir de modèles de documents au format ODT.


Nous avons cherché à améliorer cette fonctionnalité en développant un prototype d'éditeurs de fichiers qui pourra s'intégrer directement à notre application.


Les outils utilisés :


  • Langage de programmation : PHP
  • IDE : Visual Studio Code
  • Outil de Versionning : Git
  • Site de Versionning : GitHub (il faut créer un compte)
