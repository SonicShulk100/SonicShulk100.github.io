M2L Site Statique - Romain MOUROUVIN

Description du Projet :

Ce projet explique la structure générale d'un page web écrit en HTML.


Les outils utilisés :

  • Langage : HTML (HyperText Markup Language) version 5
  • IDE : Visual Studio Code
  • Outil de versioning : GitLab (Il faut créer un compte)

Compétences acquises :

  • Développer la présence en ligne de l'organisation :
    • Participer à la valorisation de l’image de l’organisation sur les médias numériques en tenant compte du cadre juridique et des enjeux économiques
    • Référencer les services en ligne de l’organisation et mesurer leur visibilité.
  • Travailler en mode Projet :
    • Analyser les objectifs et les modalités d’organisation d’un projet

Faite en HTML pure
